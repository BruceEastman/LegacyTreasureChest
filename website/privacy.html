Privacy Policy

Last Updated: March 2026

Legacy Treasure Chest is designed with a privacy-first architecture. Most information entered into the application is stored locally on the user's device and is not uploaded to central servers.

Information Stored on Your Device

The application stores information locally on the device where it is installed. This may include:

  • Item catalog information
  • Item photographs
  • Associated documents
  • Audio notes
  • Beneficiary assignments
  • Inventory organization such as sets, lots, or batches

This information remains on the device and is not transmitted to Legacy Treasure Chest servers.

AI Processing

Legacy Treasure Chest includes optional AI-assisted features that provide advisory guidance.

When these features are used, certain information may be sent to cloud services for processing, including:

  • Item photographs
  • Item descriptions

These requests are processed through the Legacy Treasure Chest AI gateway and forwarded to third-party AI services.

The system processes requests in a stateless manner and does not store inventory data on servers.

Third-Party Services

The application may use the following services:

  • Google Cloud Run
  • Google Gemini API
  • Google Places API

Server Logging

The Legacy Treasure Chest AI gateway records limited request metadata for operational purposes. This may include:

  • Anonymous device identifier
  • Request timestamps
  • Request type
  • System diagnostic information

These logs are used for debugging, monitoring service reliability, and enforcing rate limits. Server logs do not contain your household inventory data.

User Control

Users maintain full control over their data. Items, photos, and documents can be removed from the application at any time. Removing the application from the device will also remove locally stored data.

Changes to This Policy

This Privacy Policy may be updated periodically. The “Last Updated” date will reflect the most recent revision.
